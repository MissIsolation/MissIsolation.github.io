Markdown使用指南

α 对于文字的操作

** 中间的部分用粗体表示 如 我是一只喵

* 中间的部分表示为斜体 如 本喵想吃小鱼干了

~~ 中间的部分表示为删除线 如 BUG改完了

\ 后加这些特殊符号表示转义，即不产生效果只显示本身符号

β 对于段落的操作

1 几级标题就用几个#+标题文字 记得#与文字中间要加一个空格 如

看板喵 の Blog

2 > 后的一段会变为引用 如

快起床！BUG要改不完了！

喵！（炸毛）

3 * 或+ 后的一段会加标识（本喵也不知道这叫什么，反正就这么个东西）

可以按Tab键进行分级

  • 喵喵要出去玩~
    • 喵喵回来了~
  • 喵喵又要出去玩了~

4 插入表格

表格的基本格式

| 喵喵爱吃的菜 | 喵喵不爱吃的零食 |

| : ————- | : ————- |

| 小鱼干 | 橘子 |

| 虾虾 | 干果 |

Command+T 可以插入表格 (Typora)
